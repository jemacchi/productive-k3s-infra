# AWS Single Node Use Case

Basic AWS single-node infrastructure for Productive K3S.

This public use case should stay intentionally simple:

- one virtual machine
- basic network access
- SSH provisioning
- Productive K3S running in `single-node` mode

Production-grade AWS patterns such as HA, private networking, load balancers, hardened security groups, managed backups, and upgrade orchestration belong in the private `productive-k3s-infra-pro` repository.
